Arquitectura

1. Presentacion

HTML estatico con Tailwind CDN. No hay backend propio ni bundler; la carga depende del navegador y del contenido servido por GitHub Pages.

2. Feed operativo

SYSTEM.json concentra KPIs, ejecuciones recientes, workflows y actividad por hora.

3. Historico

OPERATIONS_HISTORY.json agrega snapshots diarios para comparativas y lectura de tendencia.

4. Contexto humano

DASHBOARD_CONTEXT.json formaliza proyectos, tareas y notas; Markdown queda como fallback.
